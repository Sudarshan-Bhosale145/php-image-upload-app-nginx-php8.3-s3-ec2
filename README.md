# php-image-upload-app-nginx-php8.3-s3-ec2
It is a cloud-based image upload application built using PHP 8.3 and NGINX on an EC2 instance, where uploaded images are securely stored in an Amazon S3 bucket. It demonstrates a 2-tier architecture with an optional EC2 database server, AWS SDK integration, and complete setup for deploying scalable PHP applications on AWS.
